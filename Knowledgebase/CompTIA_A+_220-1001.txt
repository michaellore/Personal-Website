	* 3.2 – Identify common connector types.
	* 3.3 – install RAM types
	* 3.4 – select, install and configure storage devices.
	* 3.5 – Given a scenario, install and configure motherboards, CPUs, and add-on cards.
	* 3.6 – Explain the purposes and uses of various peripheral types.
	* 3.7 – Summarize power supply types and features.
	* 3.8 – select and configure appropriate components for a custom PC configuration to meet customer specifications or needs.
	* 3.9 – Given a scenario, install and configure common devices.
	* 3.10 – Configure SOHO multifunction devices/printers and settings.

	* 5.3 – troubleshoot hard drives and RAID arrays.
	* 5.4 – troubleshoot video, projector, and display issues.
	* 5.5 – troubleshoot common mobile device issues while adhering to the appropriate procedures.
	* 5.6 – troubleshoot printers.
	* 5.7 – troubleshoot common wired and wireless network problems.
