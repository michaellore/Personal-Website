====== OSI Model ======

Similar to [[TCPIP Model|TCP/IP Model]] but abstracts application layer into multiple layers

Bottom - physical, connects entity to transmission media
2nd - data link - error control
3rd - network - routes info in network
4th - transport - end to end communication control
5th - session - handles non-communication issues
6th - presentation - converts info
7th - application - provides services to the application
